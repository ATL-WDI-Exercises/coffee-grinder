Similar to Ruby; however, in Javascript conditionals require parenthesis and the code is enclosed in curly-braces {}.

7. Array Functions

concat()
Joins two or more arrays
indexOf()
Search the array for an element and returns its position
join()
Joins all elements of an array into a string
pop()
Removes the last element of an array, and returns that element
shift()
Removes the first element of an array, and returns that element
push()
Adds new elements to the end of an array, and returns the new length
sort()
Sorts the elements of an array

8. Math Functions

Javascript puts Math-related functions into a Math module. Since front-end can sometimes involve geometry, trigonometry, etc it may be useful to know where to find these functions, as well as random() and round() being the most noteable.

Math.min(n1, n2, n3)
Find the lowest value in a list of arguments
Math.max(n1, n2, n3)
Find the highest value in a list of arguments
Math.random()
A random number between 0.0 and 1.0
Math.round(n)
The value of a number rounded to the nearest integer
Math.ceil(n)
The value of a number rounded up to the nearest integer
Math.floor(n)
The value of a number rounded down to the nearest integer
Math.sin(n)
The sine of a number.
Math.cos()
The cosine of a number.
Math.tan()
The tangent of a number.

Given a variable price round it to the nearest cent and store as result.
